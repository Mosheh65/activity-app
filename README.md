# מציג Activity

אפליקציית Android שמציגה בזמן אמת את ה-**Activity** (מסך) הפעיל בקדמת המכשיר — בדומה לפלט של `adb shell dumpsys activity activities`.

## הורדה והתקנה

| | |
|---|---|
| **גרסה** | 1.1|
| **Android מינימלי** | 6.0|

1. הורד את [app-release.apk](releases/app-release.apk)
2. במכשיר: אפשר **התקנה ממקורות לא ידועים**
3. פתח את הקובץ והתקן

## מה האפליקציה עושה

- מזהה את האפליקציה וה-Activity שבחזית (למשל `com.example.app/.MainActivity`) באמצעות **גישה לנתוני שימוש**
- מציגה את המידע ב**חלון צף** מעל שאר האפליקציות
- **העתקה ללוח** בלחיצה
- **אריח Quick Settings** להפעלה וכיבוי מהיר — הפאנל נסגר אוטומטית אחרי לחיצה
- **עברית / אנגלית**, מצבי תצוגה **בהיר / כהה / מערכת**

## הוספת כפתור להגדרות המהירות (Quick Settings)

ניתן להוסיף אריח קיצור לפאנל ההגדרות המהירות **ישירות מהאפליקציה**:

1. פתח את האפליקציה → **הגדרות** (⚙️)
2. גלול לסעיף **הגדרות מהירות** (מתחת להרשאות)
3. לחץ **הוסף כפתור להגדרות המהירות**
4. **Android 13 ומעלה** — אשר בדיאלוג המערכת; האריח יתווסף לפאנל
5. **גרסאות ישנות יותר** — יוצגו הוראות להוספה ידנית (עריכת QS → גרירת האריח לשורה העליונה)

האפליקציה מציגה **סטטוס** אם האריח כבר קיים בפאנל. אפשר ללחוץ שוב על הכפתור בכל עת כדי לפתוח שוב את דיאלוג ההוספה.

## הרשאות נדרשות

1. **גישה לנתוני שימוש** — לזיהוי ה-Activity הקדמי  
2. **הצגה מעל אפליקציות אחרות** — לחלון הצף  
3. **התראות** (Android 13+) — לשמירה על השירות ברקע
